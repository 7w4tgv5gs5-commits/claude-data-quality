testdb_kimball — dimensional / analytics database

Classic Kimball star schema derived from testdb via ETL. All foreign keys are enforced. Orphaned or NULL source keys are routed to Unknown sentinel members (SK = 0 / date_sk = −1) so no fact rows are lost during the load.
